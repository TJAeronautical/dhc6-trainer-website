Release history

Changelog

Desktop, mobile, live trainer, and website release notes.

Latest

Desktop v1.7.6 — Grounded climb engine

The CLIMB tab in the Performance screen now computes real climb data instead of repeating the TO/LDG outputs. Calibrated to AFM Section 5 examples: both-engine ROC at Vy (AFM Fig 5-19 anchors −10°C/2000ft/11000lb→1620 ft/min), OEI ROC at VYSE (AFM Fig 5-21 anchor +28°C/2000ft/12500lb→295 ft/min), and OEI enroute gradient (QRH 10.7 anchor +28°C/2000ft/12000lb→0.019). Negative OEI ROC is flagged in red. Also: dashboard hero card now shows the TJ Aeronautical logo; the green pill now shows the real system-group count.

Desktop v1.7.4 — Full 3D Systems Lab asset wiring

Nine system groups, real GLB models

Systems Lab extended from 6 to 9 groups: added Landing Gear/Tyres, Bleed Air/Pneumatics, and Aircraft Variants. GLB path matching corrected to align with all 19 real models in core-res — PT6A-27 cutaway, Hartzell propeller, fuel system, hydraulic pack, DHC-6 aircraft variants (wheels/floats/skis), and more. Aircraft-variant canvas height raised for wide fuselage models.

Desktop v1.7.3 — jMonkeyEngine 3D viewer enabled

Proper 3D rendering unlocked

Added the five JME3 3.6.1-stable Maven artifacts that were absent from build.gradle — the root cause of the 3D viewer not rendering. Orbit/arcball camera (drag to rotate, scroll to zoom), 4-light setup, Bloom + FXAA post-processing, auto-rotate when idle, PT6A-27 procedural fallback engine.

Desktop v1.7.1 — Logbook + score persistence

Drill scores saved locally

MCC Callout and Drill sessions now record to a local logbook via java.util.prefs.Preferences. Session counts, best %, focus-areas debrief, and a scrollable recent-attempts list with date, time, and score.

Live Trainer — full rebuild

Working in-browser trainer in the new design system

live.html is a full working trainer using site-redesign.css: Procedures, QRH, scored Flashcards, Performance (with MTOW/tailwind flags), scored Cockpit Drill, Logbook, and an interactive 3D PT6A-27 engine cross-section (Three.js, drag-orbit, click-to-learn). 25 flashcards covering PT6A-27 limits, V-speeds, fuel/oil/hydraulic/electrical systems. Three QRH items including Generator Fail. Data saved in the browser; no server required.

Website

Site redesign: site-redesign.css

All main pages use a unified design system — navy cockpit-dark background, blue/cyan accent, glass cards, responsive grid. Desktop access flow, privacy policy, and account/data deletion page. Changelog migrated from bespoke inline CSS to the shared design system.

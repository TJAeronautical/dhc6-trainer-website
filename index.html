MCC Callout Trainer

The MCC module is planned as an interactive callout trainer for multi-crew coordination. It will focus on role-based callouts, expected responses, timing, and scenario completion.

  • PF / PM role selection
  • Normal, abnormal, and emergency callout sessions
  • Step progress, timer, and score summary
  • Audio playback and response practice planned
MCC CALLOUT TRAINER SESSION 3 / 12
PM CALLS “ENGINE FIRE LEFT”
YOUR RESPONSE Tap to respond or select the correct callout.
EXPECTED FLOW Identify → Verify → Action → QRH
SESSION SCORE Accuracy, timing, and call flow summary.

PC Platform

The PC platform is planned as a larger-screen version of the DHC-6 training experience. The goal is to provide cockpit-background training panels, QRH practice, drill scoring, and MCC callout sessions for desktop users.

The first public version of this website will market the mobile app. The PC platform can later be added as either a browser-based web trainer or a downloadable Windows application.

Planned PC Features

  • Widescreen cockpit-background layout
  • Interactive QRH and MCC drill panels
  • Progress tracking and session summaries
  • Future Windows download or browser demo

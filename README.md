# DHC-6 Trainer Website

Static marketing and compliance website for the DHC-6 Trainer Android app and future PC training platform.

## Live URLs

- Website: https://dhc6-trainer.netlify.app
- Privacy Policy: https://dhc6-trainer.netlify.app/privacy.html
- Account/Data Deletion: https://dhc6-trainer.netlify.app/account-deletion.html

## Files

- `index.html` — Homepage and marketing landing page.
- `privacy.html` — Google Play-ready privacy policy.
- `account-deletion.html` — Account and associated data deletion request page.
- `robots.txt` — Search engine crawl instructions.
- `sitemap.xml` — Search engine sitemap.
- `assets/favicon.svg` — Site icon.
- `assets/site.webmanifest` — Basic web app manifest.

## Notes

This is a pure HTML/CSS website. No build step is required.

Deploy by uploading/committing these files to the `dhc6-trainer-website` GitHub repository connected to Netlify.

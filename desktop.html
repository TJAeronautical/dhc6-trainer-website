CURRENT SOURCE

Compose Desktop module

The uploaded desktop app is a Kotlin/JVM Compose Desktop source module, not a browser app. It should be built into installer packages before publishing.

VERSION

Desktop v1.6.9

The desktop module declares packageVersion 1.6.9 and targets MSI, EXE, DEB and DMG native distributions.

USE CASE

PC / classroom platform

Use desktop for larger panels, procedure review, instructor display, MCC callouts and deeper systems work.

Install guidance

  • Publish checksums beside every installer.
  • Keep a short known-issues list for each desktop release.
  • Do not ask users to download the source zip as the desktop app.
  • Use GitHub Releases or a signed cloud download for installer delivery.

Desktop value proposition
